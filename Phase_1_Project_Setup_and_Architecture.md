## Antigravity Prompt

Create a full-stack AI-based travel recommendation system using: - React
(Frontend) - Flask (Backend API) - MongoDB - scikit-learn for ML Follow
modular structure. Do not mix frontend and backend logic.

## Rules for Antigravity

-   Follow clean architecture.
-   No hardcoded values unless specified.
-   Maintain modular structure.
-   Follow REST API conventions.
