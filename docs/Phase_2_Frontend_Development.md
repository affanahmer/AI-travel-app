# Phase 2 -- Frontend Development

## Objective

Develop React-based UI for user interaction.

## Features

-   Home page with search bar
-   Destination cards
-   Guest access
-   Travel suggestions
-   Login/Register pages
-   Profile page
-   Admin dashboard UI

## Requirements from Document

-   Dynamic recommended cards
-   Travel suggestions section
-   Optional login for saving trips

## Antigravity Prompt

Build responsive React UI with: - Search input parsing - Cards rendered
from API - Conditional rendering for guest vs logged-in users - Separate
Admin dashboard routes

## Rules

-   Use reusable components
-   Maintain responsive layout
-   Use proper state management
